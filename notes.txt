How to Install a Virtual Environment using Venv

1. pip install virtualenv
2. py -m venv env #for current version
2a. python<version> -m venv <virtual-environment-name> #to pick version from

Activate Environment:
    env/Scripts/activate.bat //In CMD
    env/Scripts/Activate.ps1 //In Powershel

How to Deactivate a Virtual Environment:
     ~ deactivate

Source: https://www.freecodecamp.org/news/how-to-setup-virtual-environments-in-python/
